# NFL INJURY STATISTICS

## Project Overview
This project looked at the injury statistics from the NFL over the past 12 years. I generated numerous graphs that give valuable insights to injury data. From this project, I expanded on my skills in handling large-scale data sets, data cleaning and manipulation, as well as data analytics. The goal of this project was to find what postions are most suscepcible to injury, what injuries are the most common, and how the NFL is performing on the topic of injury prevention. From my findings, we can take away what needs to change for these athletes in terms of recovery, technology, and field conditions.
